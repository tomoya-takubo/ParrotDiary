# ぱろっとだいありーのテーマ

```
「毎日をちょっとだけ楽しく記録する」
自己成長に関心のある全年齢層向けのWeb日記アプリ、ぱろっとだいありー（ParrotDiary）。

このアプリは、気軽に続けられる3行日記の記録を中心に、日々の思いや出来事を蓄積していけるシンプルな仕組みが特長です。
感じたことは感情タグで残せて、よく使うタグは上位候補として表示されるため、記録がどんどん自分らしくなっていきます。

日記を書くたびに登場するのは、ガチャで集めた466種類のPartyParrotたち。
日記の“象徴”としてお気に入りのパロットを選び、その日の記録に彩りを添えられます。

また、カレンダービューでの振り返りや、検索・絞り込み機能も搭載。
日々の記録が積み重なることで、ちょっとした変化や自分の傾向に気づくことができます。

書くことそのものを楽しみたいあなたへ。
ぱろっとだいありーは、「無理なく続ける」ためのやさしくポップな日記アプリです。
```
